HTML:

HTML: HiperText Markup Language (Linguagem de marcação de Hipertexto), é o componente básico da web (o navegador faz a leitura do arquivo e renderiza para o usuário) e foi criada pelo britânico Tim Berners-Lee. Os sites possuem diferentes distribuições e tamanhos de fontes, como por exemplo em títulos e parágrafos, essa organização é realizada através das Tags utilizadas no HTML.

Tags:

São códigos que definem toda a estrutura da página.

Principais tags:

  • html: Tag principal, tudo que estiver dentro dela, faz parte do arquivo HTML
  • head: Não é visivel para o usuário.
  • body: Corpo que fica visivel para o usuário

    • header Cabelhaço do body
    • footer Rodapé do body

Tags de texto:

  • i: itálico
  • strong: negrito
  • u: sublinhado
  • h1: Titulo (maior tamanho de fonte)
  • h2, h3, h4, h5 e h6: Titulos (diminuição da fonte de acordo com o número do h)
  • p: Parágrafo
  • blockquote: Citação

Tags de lista:

  • ol: Lista ordenada
  • ul: Lista não ordenada
  • li: Itens da lista

Tags de links:

  • a: Hiperligação

    • Atributos target:

    • _blank: Abre o link em outra aba
    • _ self: Abre o link na mesma aba
    • title: Aparece uma caixa de texto quando passa o mouse sobre o link
